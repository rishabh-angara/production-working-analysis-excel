# production-working-analysis-excel
# 📊 Production Work Analysis Dashboard (Excel)

## 📌 Project Overview
This project analyzes production performance using an interactive Excel dashboard.  
It highlights key KPIs such as total production, downtime percentage, operator performance, and daily/shift output trends.

## 🎯 Objectives
- Identify production patterns and bottlenecks  
- Analyze operator-wise and shift-wise performance  
- Track downtime and defect trends  
- Provide actionable insights for improving efficiency
  
** ** 📂 Dataset Description**
The dataset contains production workforce information, including:

Employee Name
City
Division
Job Title
Gender
Planned Working Hours
Worked Hours
Overtime Hours
Earnings
Unworked Hours
Unworked Amount

📈 Key Performance Indicators (KPIs)

The dashboard summarizes important production metrics using KPI cards:
| KPI                 | Description                                          |
| ------------------- | ---------------------------------------------------- |
| **Unworked Amount** | Total financial loss due to unworked hours.          |
| **Unworked Hours**  | Total hours not worked compared to planned schedule. |
| **Amount Earned**   | Total earnings generated by employees.               |
| **Planned Hours**   | Total scheduled working hours.                       |
| **Worked Hours**    | Actual completed working hours.                      |
| **Overtime Hours**  | Total overtime worked beyond planned hours.          |

**📊 Dashboard Components**
1️⃣ City-wise Earnings Analysis

Displays earnings generated by each city.
Identifies top and lowest-performing locations.
Helps management compare regional performance.

Chart Used: Clustered Column Chart

**2️⃣ Division-wise Work Distribution**

Compares:

Planned Hours
Worked Hours

across divisions such as:

Photography
Supporting
Management

Purpose

Measure operational efficiency.
Identify underutilized departments.
Compare workload distribution.

Chart Used: Clustered Bar Chart

**3️⃣ Earnings by Division**

Shows the percentage contribution of each division toward total earnings.

Example:

Photography
Supporting
Management

Purpose

Identify the highest revenue-generating division.
Compare departmental contribution.

Chart Used: Pie Chart

**4️⃣ Job Title and Gender Distribution**

Visualizes workforce composition based on job roles, including:

Cameraman
Assistant
Coordinator

Purpose

Understand workforce allocation.
Analyze employee role distribution.

Chart Used: Pie Chart

**5️⃣ Employee-wise Earnings**

Displays earnings generated by individual employees.

Purpose

Identify top performers.
Compare employee contributions.
Support performance evaluation.

Chart Used: Line Chart with Data Labels

**6️⃣ Overtime Worked Employees**

Shows employees who have accumulated the highest overtime hours.

Purpose

Detect excessive overtime.
Improve workforce planning.
Reduce employee fatigue.
Balance workloads.

Chart Used: Column Chart

**🎛 Interactive Features**

The dashboard includes dynamic filters (Slicers) for:

City
Division

These slicers allow users to instantly filter all charts and KPIs for detailed analysis.

## 🧠 Insights Derived
- Tokyo generated the highest earnings among all cities.
-Photography division contributes the majority of overall earnings.
-Worked hours slightly exceed planned hours, indicating additional workforce utilization.
-Certain employees consistently generate higher earnings.
-A small group of employees accounts for most overtime hours.
-Unworked hours and associated financial loss highlight opportunities for better workforce scheduling.
